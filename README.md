🎓 Student Result System (Python)

A simple Python program that takes student details, calculates total marks and percentage, and determines the result (Pass/Fail) along with the grade.

---

📌 Features

- Input student name and roll number
- Accept marks for 3 subjects
- Calculate:
  - Total Marks
  - Percentage
- Determine:
  - Pass or Fail status
  - Grade based on percentage
- Clean and user-friendly output

---

🧮 Grading Criteria

| Percentage       | Grade |
|----------------|------|
| 80 and above   | A+   |
| 70 – 79        | A    |
| 60 – 69        | B    |
| 50 – 59        | C    |
| Below 50       | Fail |

---

🛠️ Technologies Used

- Python 3
- Basic programming concepts:
  - Variables & Data Types
  - Input/Output
  - Arithmetic Operators
  - Conditional Statements (`if`, `elif`, `else`)
  - Type Casting
  - f-strings

---

▶️ How to Run

1. Make sure Python is installed on your system.
2. Download or copy the code into a file named:

```

student_result.py

````

3. Open terminal or command prompt and run:

```bash
python student_result.py
````
